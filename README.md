# Powerplant
This is my 1st Deep Learning project where I have built an ANN(Artificial Neural Network) Model for Regression where data is evaluated with the help of the concept of Deep Learning.
Basically in this project I have used a powerplant dataset where there is over 9000+ values and 4 features and had to make evaluation of the model using ANN Regression.
There were several factors namely: temperature, vacuum, humidity and pressure as input. On the basis of these features I had to find out the produced energy.
I faced lot of difficulties in Evaluating the model but hopefully figured all out.
